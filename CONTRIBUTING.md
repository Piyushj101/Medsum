# Contributing to MEDSUM

Thank you for your interest in contributing to MEDSUM.

## Getting Started

1. Fork the repository.
2. Clone your fork.
3. Create a new feature branch.
4. Make your changes.
5. Test your changes.
6. Commit using meaningful commit messages.
7. Submit a Pull Request.

## Coding Standards

- Follow consistent coding practices.
- Write clean and readable code.
- Document major features and API changes.
- Keep commits focused on a single task.

## Reporting Issues

Please include:

- Description of the issue
- Steps to reproduce
- Expected behavior
- Screenshots (if applicable)

## Contributors

This project was developed as part of a college team project. All contributors are acknowledged based on their respective contributions.
